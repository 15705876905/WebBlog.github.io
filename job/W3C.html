# 规范标准

  • 需要声明（DOCTYPE）

    来说明使用的是XHTML还是HTML的版本

  • 定义语言编码

  • Js必须要用

    <script language="javascript" type="text/javascript">

    开头定义

  • CSS必须要用

    <style type="text/css">

    开头来定义

  • 等号代替注释里的虚线

  • 标签元素和属性名小写

  • 属性必须用引号括起来

  • 所有特殊符号用编码表示

    &nbsp空格、&alt<、&gt>

  • 标记必须又结束标记

  • 合理嵌套

  • 添加alt属性图片

  • form添加label

# 结论

  • 标签规范提高搜索引擎对页面的抓取效率，SEO优化
  • 尽量使用外联js和css，提高渲染速度
  • 样式尽量少用行间样式表，使结构与表现分离，标签的id和class等属性命名要做到见文知义，标签越少，加载越快，用户体验提高，代码维护简单，便于改版

# BFC规范

# 决定了元素如何对其内容进行布局，以及与其他元素的关系和相互作用。

盒子从顶端开始垂直地一个接一个地排列，盒子之间垂直的间距是由 margin 决定的。在同一个 BFC 中，两个相邻的块级盒子的垂直外边距会发生重叠。

# 创建 BFC

  • 根元素或包含根元素的元素
  • 浮动元素 float ＝ left | right 或 inherit**（≠ none）**
  • 绝对定位元素 position ＝ absolute 或 fixed
  • display ＝ inline-block | flex | inline-flex | table-cell 或 table-caption

# Cookie

始终在同源的http请求中携带，即使不需要。设置的过期时间之前一直有效，即使窗口关闭。

        function setCookie(name, value, day) {
            let d = new Date();
            d.setTime(d.getTime() + (day * 24 * 60 * 60 * 1000));
            let expires = `expires${d.toUTCString()}`;
            document.cookie = `${name}=${value};${expires};path=/`;
        }
        function getCookie(name) {
            let cookies = document.cookie;
            cookies = cookies.split(';');
            for (i of cookies) {
                i = i.trim();
                if (i.indexOf(`${name}=`) != -1) {
                    return (i.substring((`${name}=`).length, i.length));
                }
            }
            return "null";
        }
        setCookie('name', 'ckf', 1);
        setCookie('id', 1231235, 2);
        console.log(getCookie('n'));

# localStorage

存储持久数据，游览器关闭后不会丢失，除非主动删除。

window.localStorage
保存数据语法
localStorage.setItem("key", "value");
读取数据语法：
var lastname = localStorage.getItem("key");
删除数据语法：
localStorage.removeItem("key");

# iframe

  • iframe会阻塞主页面的Onload事件
  • 搜索引擎的检索程序无法解读这种页面，不利于SEO
  • iframe和主页面共享连接池，而浏览器对相同域的连接有限制，所以会影响页面的并行加载

# Canvas和SVG

svg是一块画布，放大会失真或者锯齿。

# rgba()和opacity的透明效果有什么不同？

  • rgba()和opacity都能实现透明效果，但最大的不同是opacity作用于元素，以及元素内的所有内容的透明度，
  • 而rgba()只作用于元素的颜色或其背景色。（设置rgba透明的元素的子元素不会继承透明效果！）

# 严格模式

就是让js在严格模式下运行，使js编码规范化，消除一些不合理的操作，不严谨的地方。

  • 变量先声明，后使用
  • 函数的参数不能有同名属性
  • 不能使用with
  • 禁止this指向全局
